Template Information

Description :

  • Chaveiro is a creative, elegant, modern, clean and Responsive HTML5/CSS3 Locksmith Template based on Bootstrap with SASS. Compatible with Desktop, Laptop, Tablet, Mobile or any device. Chaveiro is simple and clean design. All files are clearly organised. We believe it will be easy to use and edit them.

Content Mapping :

  • Template root has 12 .html files and 3 folders
  • Stylesheets (CSS files) of this template are placed in assets/css folder. style.css file for template main/custom style sheet.
  • All Images are in assets/images folder.
  • And all Javascripts files are in assets/js folder.

HTML Structure

The template is based on Bootstrap 4 the world's most popular front-end component library for developing responsive, mobile-first projects on the web.

Bootstrap includes a responsive, mobile first fluid grid system that appropriately scales up to 12 columns as the device or viewport size increases. It includes predefined classes for easy layout options, as well as powerful mixins for generating more semantic layouts.

Create a .container (fixed-width) or .container-fluid (full-width) for proper alignment and padding, then create .row and add columns with appropriate device name (xs, sm, md, lg) and grid number (2, 4, 6, 12).

For full documentation, please visit Bootstrap 4 site: https://getbootstrap.com/docs/4.1/getting-started/download/

Color Variation

We have predefined thirteen color and also black and white color.

  • First color class name is primary-color
  • Second color class name is second-color
    • and also
  • Black color class name is black-color
  • White color class name is white-color
  • White color class name is dark-color
  • White color class name is facebook-color
  • White color class name is twitter-color
  • White color class name is dribbble-color
  • White color class name is instagram-color
  • White color class name is skype-color
  • White color class name is linkedin-color
  • White color class name is rss-color
  • White color class name is pinterest-color
You can use these class in any div or section to get color according to this class. You can change color code. Try yourself to use/change classname
N.B.: Please observe carefully our HTML code to see color variation output. Such as index2.html
If you are familiar or know SASS, then you can do easily color compile in variable.scss file. Otherwise change color in style.css file

Google Fonts

Open style.css in any text editor from css folder. And see imported Google font link. 

@import url('https://fonts.googleapis.com/css?family=Mulish:wght@400;500;600;700;800&family=Nunito+Sans:wght@400;600;700;800;900&display=swap');

We are using Google font and other open source font. You can change here font link that you want. Then need to change font-family at that div / tag/ section class in style.css file.

Icons

We are using IonIcons

These font css file in this directory: css\fonts\ionicons\css\ionicon.css

Use icon class name from above css files.


HTML Markup: 

<i class="ion-ios-send"><i>

You can use another icon you like. Then keep font in fonts family. And keep icon font css in css folder like ionicon.css

Then use font in HTML as per using method of that font.

If you need more information about these icon sets, please visit this sites: IonIcons
